# Описание проекта

Проект представляет собой первую версию мобильного приложения туристического гида. Логика приложения строится вокруг карты, списка историко-культурных мест, карточки места, пользовательских отзывов, фотографий, жалоб, маршрутов, достижений и заданий.

Главная цель структуры — подготовить основу для реализации реального приложения, а не кликабельного макета. Все данные должны проходить через Room/SQLite и слой Repository.
